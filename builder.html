  B U I L D E R  

Builder is an application designed to assist people stop procrastinating and start getting things done. Builder does this by encouraging collaboration with friends, family and workmates and rewarding you as you achieve your goals.

With Builder, you will be asked to create and manage goals. As you complete the tasks you set for yourself, you will be building an amazing world.

Here we can see the Builder home menu. The screen currently displays a goal with its required tasks and an icon showing the what the reward for each completed task.

This screen also shows the options to

  • Add a buildings to their world
  • Invite a friend
  • Set a new task and set a new goal
  • Create a new goal

This mockup shows a user interface screen that can be selected from the main screen of Builder. From this screen, the user can see the goal they are currently working on as well as the tasks that are required to meet their goal.

The user will also keep track of any additional notes.

This screen will also have the option to

  • Set the goal as the current focus
  • Add additional tasks to the goal
  • Mark tasks as completed

This screen shows the city as it is being built in Builder. The user will see all of their hard work displayed as an aesthetically pleasing city, filled with buildings that are representations of their completed tasks and goals.
